# Simon Data

Simon Data is an AI-first customer data platform (CDP) purpose-built to increase campaign performance through faster, more precise segmentation and personalization. It combines customer data with contextual signals to enable 1:1 personalization at scale through AI agents that automate data preparation and campaign execution.

## APIs

- **Simon Signal Event Ingestion API** — POST events (authentication, cart, transactions, page views, product views, registration, search, waitlist, custom) to `https://dev.simonsignal.com/events/v1/collect` using a shared partner secret.
- **Contact API** — Retrieve and manage customer profile records and traits for segmentation workflows.
- **Real-Time Content API** — Premium feature that fetches external data at message send time for dynamic personalization across channels.
- **Outbound Webhooks API** — Push segment activation and event data to downstream marketing tools and data warehouses.
